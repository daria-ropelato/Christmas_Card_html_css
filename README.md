# 🎄 Merry Christmas CSS Animation Card

A fun and festive CSS animation of a Christmas tree with glowing lights and a pulsing star, combined with a typewriter animation of "Merry Christmas!" text. Built entirely with **HTML and CSS** — no JavaScript required.

---

## ✨ Features

- **CSS Tree** with three triangle layers (small → large)  
- **Random glowing Christmas lights** distributed across the tree  
- **Pulsing 5-point star** on top with orange glow effect  
- **Typewriter text animation** for "Merry Christmas!" that appears and disappears slowly  
- Fully responsive card container  
- Smooth animations and micro-interactions suitable for UI/UX portfolio demonstrations

## ✨ Tools & Technologies

- HTML5 — structure and layout of the card  
- CSS3 — animations, gradients, shapes, keyframes  
- Flexbox — for centering and layout alignment  
- CSS clip-path — to create the star shape  
- GitHub Pages — hosting live demo  

---

## ✨ How It Works

- **Tree**: built with CSS triangles of varying sizes  
- **Lights**: CSS keyframe animations with different `animation-delay` and `opacity` to create a random glow effect  
- **Star**: pulsing `scale()` animation with box-shadow glow  
- **Text**: typewriter effect using `steps()` and `width` animation
